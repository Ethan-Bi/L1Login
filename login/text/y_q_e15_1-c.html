步驟.1 困惑的瑪勒巴 ：

在妖精森林裡找到瑪勒巴(33053, 32315)，和他交談後會給你瑪勒巴的信。
並且請玩家將信拿去給阿拉斯。

任務目標：
跟瑪勒巴接洽任務，並探聽下一個步驟

相關物品：
瑪勒巴的信 x 1

步驟.2 著急的阿拉斯 ：

在妖精森林的眠龍洞穴入口旁，可以找到阿拉斯，並且將瑪勒巴的信交給他。
之後，他會請求玩家們幫他蒐集遠征隊的遺物，並給你阿拉斯的護身符。

任務目標：
跟阿拉斯接洽任務，並探聽下一個步驟

相關物品：
阿拉斯的護身符 x 1

步驟.3 遠征隊的遺物 ：

在眠龍洞穴 1樓和眠龍洞穴 2樓狩獵汙濁獨眼巨人，可以獲得遠征隊的遺物。
打開遠征隊的遺物可以獲得汙濁裝備(全部6種)，將汙濁裝備拿去給阿拉斯，他會給予你同樣款式的遠征隊裝備。

汙濁的弓 → 遠征隊弓
汙濁的金甲 → 遠征隊金甲
汙濁的腕甲 → 遠征隊腕甲
汙濁的頭盔 → 遠征隊頭盔
汙濁的鋼靴 → 遠征隊鋼靴
汙濁斗篷 → 遠征隊斗篷

當湊齊六種遠征隊裝備，在與阿拉斯交談以及歸還阿拉斯的護身符。
之後他會給予阿拉斯的信，並且跟你說如果想要修好這些裝備就去找瑪勒巴。

注意事項：
兩樓層所獲得的遠征隊的遺物，能夠得到的汙濁裝備不相同，因此要湊齊整套汙濁裝備，需要兩個樓層都蒐集

任務目標：
湊齊遠征隊裝備，並獲得阿拉斯的信

相關怪物：
Lv.14 汙濁獨眼巨人
Lv.18 汙濁獨眼巨人

相關物品：
汙濁的弓 x 1
汙濁的金甲 x 1
汙濁的腕甲 x 1
汙濁的頭盔 x 1
汙濁的鋼靴 x 1
汙濁斗篷 x 1
阿拉斯的信 x 1

步驟.4 修復裝備 ：

將阿拉斯的信交給瑪勒巴，之後瑪勒巴就會跟你說修復這些裝備需要那些材料。
接下來只要蒐集好材料就能修復遠征隊裝備，並且修好的裝備會顯示出自己的名子，證明修復者的身份。

遠征隊弓 → ID.妖精弓
遠征隊金甲 → ID.妖精金甲
遠征隊腕甲 → ID.妖精腕甲
遠征隊頭盔 → ID.妖精頭盔
遠征隊鋼靴 → ID.妖精鋼靴
遠征隊斗篷 → ID.妖精斗篷

任務目標：
將阿拉斯的信交給瑪勒巴，即可開始蒐集材料修復裝備

相關物品：
ID.妖精弓 x 1
ID.妖精腕甲 x 1
ID.妖精金甲 x 1
ID.妖精頭盔 x 1
ID.妖精鋼靴 x 1
ID.妖精斗篷 x 1
